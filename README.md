# HotPen AI 🖊️

> AI 驱动的热点内容智能生产平台

## 功能特性

- 🔥 **多领域热点抓取** - 科技、财经、娱乐、社会、AI 五大领域
- 🔍 **深度研究** - 多轮检索 + 学术论文引用
- ✍️ **智能撰写** - 公众号长文 + 小红书文案
- 🚀 **一键发布** - 多平台发布支持

## 项目结构

```
hotpen-ai/
├── docs/                    # 项目文档
├── agents/                  # Agent 配置
│   └── trackers/           # 热点抓取 Agents
├── code/                   # 源代码
│   ├── backend/
│   ├── frontend/
│   ├── test/
│   └── quality/
└── README.md
```
